# FAQ

## Are these official OpenAI API keys?

No. These are APIMaster.ai voucher codes for OpenAI-compatible API access. They are not official OpenAI API keys or OpenAI account credentials.

## Is this project affiliated with OpenAI?

No. This repository is not affiliated with OpenAI.

## How do I redeem a voucher?

The original free voucher drop is sold out. It was fully claimed within 2 hours after launch on June 18, 2026.

You can still open APIMaster.ai, create an account, and use the first top-up discount for new users.

## Why does the project mention "OpenAI API key"?

Many developers search for OpenAI API key access when they need an OpenAI-compatible endpoint. APIMaster.ai provides compatible API access through its own API key system.

## What if a voucher has already been used?

All public voucher codes have already been claimed. APIMaster.ai currently offers a first top-up discount for new users.

## Can I use the OpenAI SDK?

Yes. Use the official OpenAI SDK and replace the base URL with the APIMaster.ai endpoint from your APIMaster console.
